Information for developers

Paradrop is an open source software platform that transforms Wi-Fi routers into smart devices using state of the art techniques related to virtualization.

We leverage core technologies like Snappy Ubuntu and Docker to enable applications to exist on Wi-Fi routers within the home. This has enabled us to create an app-store ecosystem for Internet services. With an Internet-connected, active router, and a physical footprint in the home, the possible applications are virtually endless. To get a few ideas of applications, please see our example applications.

Local Computing

The world of cloud computing revolves around a model of “location doesn’t matter”. Unless you are talking about micro or milliseconds, if you launch your application, it really doesn’t matter if it runs in a server farm in Oregon or in Iowa.

But what if it did? Imagine you could write an application that physically existed within someone’s home that was always connected to the internet, powered on 24/7, and didn’t require extra hardware? Or an application where the program knew it was at 82nd and Broadway in NYC.

Having knowledge of these locations and the assets physically nearby opens up a new world of local computing. And that world is powered by Paradrop. Paradrop provides this ability, whether this is through one of our several city-wide deployments, or within a user’s home.

City-Wide Deployments

We are partnering with several cities nationwide to provide outdoor Wi-Fi Access Points powered by Paradrop in the downtown areas. We want developers to leverage our platform by providing value added services to these cities. Imagine if outdoor sensor data could be aggregated and analyzed locally without the huge infrastructure requirements of typical city-wide deployments. What if businesses could advertise to customers based on their location as they connect to the Wi-Fi, without installing an app on the phone? All of these applications and more are possible through our city-wide Paradrop deployments. Stay tuned for our first locations to be announced very soon.
