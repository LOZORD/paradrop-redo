Information for developers

Paradrop is an open source software platform that transforms Wi-Fi routers into smart devices using state of the art techniques related to virtualization.

We leverage core technologies like Snappy Ubuntu and Docker to enable applications to exist on Wi-Fi routers within the home. This has enabled us to create an app-store ecosystem for Internet services. With an Internet-connected, active router, and a physical footprint in the home, the possible applications are virtually endless. To get a few ideas of applications, please see our example applications.
