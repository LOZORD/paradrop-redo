Description of Services

Paradrop is a software platform that enables virtual applications to install and exist on WiFi router hardware. It enables simplified control over Internet-based services, which exist in the home and business today. Using Paradrop allows any developer to quickly and easily create applications for smart home innovation. Paradrop also allows configuration and control over the Wi-Fi router for usage as a router within the home.
